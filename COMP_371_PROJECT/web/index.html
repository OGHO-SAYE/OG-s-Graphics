Project Description

This project aims to develop a computer graphics application using OpenGL to render a football field scene with various lighting effects, including Blinn-Phong and Phong shaders. The scene includes a movable camera to view the field from different angles. The scene comprises approximately 441.3k vertices, providing detailed visualization. .
